Современные веб-технологии

Можно выделить два основных типа веб-технологий. Первый тип — frontend, который отвечает за внешний вид сайта и доступен в браузере пользователя. Второй тип веб-технологий — backend, который работает на сервере и используется для обработки данных, его действия обычно скрыты от пользователя. Далее речь пойдет о frontend-технологиях.

Оглавление

  • HTML
  • CSS
  • JavaScript

Frontend

Под понятием frontend подразумевается разработка видимого для пользователя интерфейса и функций, благодаря которым пользователь может взаимодействовать с интерфейсом. В основе frontend-разработки лежат три технологии: HTML, CSS и JavaScript.

  • HTML

    HTML — это язык разметки, который используется для структурирования и отображения веб-страницы и её контента. HTML-документ состоит из элементов, которые оборачивают различные части контента, чтобы заставить их отображаться или действовать определённым образом. HTML-элементы обозначаются с помощью тегов, они могут быть парными и непарными.

    Главными частями HTML-элемента являются:

    1. Открывающий тег (Opening tag) указывает, где элемент начинается
    2. Закрывающий тег (Closing tag) указывает, где элемент заканчивается
    3. Контент (Content) — содержимое HTML-элемента
    4. Элемент (Element), состоящий из тегов и контента

    HTML-элементы также могут иметь атрибуты, которые содержат дополнительную информацию об элементе. В данном случае используется атрибут class со значением «editor-note». Класс позволяет дать элементу идентификационное имя, которое может позже использоваться, чтобы обращаться к элементу.

  • CSS

    CSS — это язык таблицы стилей, который позволяет применять стили выборочно к элементам веб-страницы. В отличие от HTML, который служит для определения структуры и семантики содержимого, CSS отвечает за его внешний вид и отображение. Например, с помощью CSS можно изменять шрифт, цвет, размер, межстрочный интервал, разделять содержимое на колонки, а также добавлять анимацию и другие декоративные элементы.

    Набор правил CSS включает в себя:

    1. Селектор (Selector) выбирает HTML-элемент с соответствующим именем
    2. Объявление (Declaration) — одно из правил, которое указывает, что и как нужно стилизовать
    3. Свойство (Property), которое нужно стилизовать
    4. Значение свойства (Property value) из множества возможных вариантов для данного свойства

  • JavaScript

    JavaScript — это язык программирования, который применяется к HTML документу, и может обеспечить динамическую интерактивность на веб-сайтах. Например, отклик при нажатии кнопок или при вводе данных в формы, динамические стили, анимация.
